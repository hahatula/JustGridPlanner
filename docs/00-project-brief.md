# Project Brief

This is a private iPhone app for planning Instagram profile grid.

The app is for personal use only:

- Not published to App Store
- Not sold
- Not distributed
- Built only for my own iPhone

Main goal:

Help me preview how future posts and reels will look in my Instagram grid before I publish them.

Core idea:

The app shows already-posted Instagram media as locked items and allows me to add local planned images from my iPhone gallery. Planned items can be reordered and removed.

Non-goal:

This app does not publish to Instagram.
